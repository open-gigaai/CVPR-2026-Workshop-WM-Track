# Simulator

This project is a modified version of [RoboTwin](https://github.com/RoboTwin-Platform/RoboTwin). It takes actions from **Piper robotic arms** as input and outputs RGB replay videos with black backgrounds.


## Environment Setup

For environment setup, please refer to: [Instruction](https://robotwin-platform.github.io/doc/usage/collect-data.html)
